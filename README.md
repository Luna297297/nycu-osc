# nycu-osc
Self-study implementation of the NYCU Operating System Capstone (OSC) labs on Orange Pi RV2 (RISC-V).

## Enviroment
### Host
- MacBook Air M1 (Apple Silicon)
- macOS

### Guest
- Ubuntu Server 22.04 LTS (ARM64)
- UTM (Apple Virtualization)

### Development Tool
- VS Code Remote SSH
- Git
- GCC / G++
- CMake
- GDB
